    [state state]]) → (is-a?/c MT19937%)
  seed : integer? = random-number
  state : vector? = default-state
Creates an instance of MT19937 with the provided seed or uses crypto-random-bytes to generate a random seed.

A malicious state can be injected at the start to clone another instance of MT19937. Otherwise the default-state is calculated from the seed.

5.2 RSA

 (require "util/rsa.rkt") package: base

procedure

(primegen [size]) → integer?

Generates a random prime number of size bytes, using probabilistic primality testing. prime? is already provided, and I believe it uses Miller-Rabin or at least something similar. Candidates are generated using crypto-random-bytes, with the MSB and LSB set to 1.

procedure

Generates the public, private key pair for RSA using the provided value for e. Each key is a pair, where the second value is n. (ie, the output is [e,n], [d,n])
