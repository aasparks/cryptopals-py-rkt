 (require "util/aes.rkt") package: base

procedure

(aes-128-encrypt txt    
  key    
  [iv-or-nonce    
  #:mode mode?]) → bytes?
  txt : bytes?
  key : bytes?
  iv-or-nonce : (or/c bytes? integer?) = 0
  mode? : symbol? = 'ECB
Performs AES-128 encrypt on txt under the provided mode. Valid options for #:mode are 'ECB, 'CBC, or 'CTR. Lower-case modes are accepted as well (i.e. 'ecb). For 'CBC and 'CTR modes, the default iv is (make-bytes 16 0) and the default nonce is 0.
